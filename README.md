# React slider

## How to run

first install node_modules folder.
```sh
npm install
```

to run development build with local server
```sh
npm run start
```

For production build
```sh
npm run build
```

### Versions

| Plugin | Version |
| ------ | ------ |
| React | 16.3.1 |
| Webpack | 4.5.0 |
